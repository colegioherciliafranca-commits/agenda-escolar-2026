# Agenda Escolar 2026

Agenda online interativa para o Colégio Estadual Profª Hercília F do Nascimento - EFMP, Mangueirinha - PR.

## 🎯 Funcionalidades

- 📅 **Calendário interativo** com todos os meses de 2026
- 🎨 **Cores oficiais** da escola (feriados, aulas, conselhos, etc.)
- ✍️ **Reuniões inline** - clique e digite diretamente no dia
- 📱 **Design responsivo** para celulares e tablets
- 💾 **Dados salvos localmente** no navegador
- 🌐 **Acesso online** via GitHub Pages

## 🚀 Como Usar

### Online (Recomendado)
Acesse: https://[seu-usuario].github.io/agenda-escolar-2026/agenda_escolar.html

### Local
1. Baixe os arquivos
2. Abra `agenda_escolar.html` no navegador

## 📋 Como Adicionar Compromissos

1. **Clique em qualquer dia** do calendário
2. **Digite a reunião** (ex: "Reunião as 14hs")
3. **Pressione Enter** ou clique fora para salvar
4. **Clique na reunião** para editar
5. **Clique em ×** para excluir
