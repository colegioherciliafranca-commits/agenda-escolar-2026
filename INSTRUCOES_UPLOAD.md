# Instruções para Upload Manual no GitHub

## 🎯 Repositório Alvo
**URL**: https://github.com/colegioherciliafranca-commits/agenda-escolar-2026

## 📁 Arquivos para Upload

### ✅ Arquivos Essenciais (UPLOAD OBRIGATÓRIO):
1. **agenda_escolar.html** (35KB) - A agenda completa
2. **README_AGENDA.md** → **README.md** (renomear) - Documentação
3. **DEPLOY.md** (4KB) - Guia de deploy

## 🚀 Passo a Passo Detalhado

### 1. Acessar o Repositório
1. Abra: https://github.com/colegioherciliafranca-commits/agenda-escolar-2026
2. Você verá "This repository is empty"

### 2. Fazer Upload dos Arquivos
1. Clique no botão **"Add file"** → **"Upload files"**
2. **Arraste os arquivos** para a área de upload:
   - `agenda_escolar.html`
   - `README_AGENDA.md`
   - `DEPLOY.md`

### 3. Renomear README
1. Após upload, encontre `README_AGENDA.md`
2. Clique no ícone de lápis ✏️ para editar
3. Mude o nome para `README.md`
4. Clique **"Commit changes"**

### 4. Verificar Upload
Após upload, você deve ver:
- ✅ agenda_escolar.html
- ✅ README.md (renomeado)
- ✅ DEPLOY.md

## 🔧 Ativar GitHub Pages

### 1. Ir para Settings
1. No repositório, clique em **Settings** (menu superior)
2. No menu esquerdo, clique em **Pages**

### 2. Configurar Deploy
1. **Source**: Selecione **"Deploy from a branch"**
2. **Branch**: Selecione **"main"**
3. **Folder**: Selecione **"/ (root)"**
4. Clique **"Save"**

### 3. Aguardar Deploy
- Aguarde 1-2 minutos
- Status aparecerá: "Your site is published at..."
