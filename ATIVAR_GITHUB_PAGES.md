## Configurando GitHub Pages

### 1. Acessar Settings
1. **Abra**: https://github.com/colegioherciliafranca-commits/agenda-escolar-2026
2. **Clique** na aba **"Settings"** (menu superior)

### 2. Navegar até Pages
1. **Menu esquerdo**: clique em **"Pages"**
2. Você verá: "Build and deployment"

### 3. Configurar Source
1. **Source**: selecione **"Deploy from a branch"**
2. **Branch**: selecione **"main"**
3. **Folder**: selecione **"/ (root)"**
4. **Clique** em **"Save"**

### 4. Aguardar Deploy
- **Aguarde 1-2 minutos**
- Status aparecerá: "Your site is published at: [URL]"

## Se Tiver Problemas

### Problema 1: Pages não aparece
- **Solução**: Verifique se você tem permissões de admin no repositório

### Problema 2: Site não carrega
- **Solução**: Aguarde mais 2-3 minutos (deploy pode demorar)

### Problema 3: Erro 404
- **Solução**: Verifique se o nome do arquivo está correto: `agenda_escolar.html`
