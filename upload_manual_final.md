# Upload Manual - Última Tentativa

## 🚀 Como Fazer Upload Agora

### Opção 1: Upload Manual (Recomendado)
1. **Acesse**: https://github.com/colegioherciliafranca-commits/agenda-escolar-2026
2. **Clique em**: "Add file" → "Upload files"
3. **Arraste os arquivos**:
   - `agenda_escolar.html`
   - `README_AGENDA.md` (depois renomeie para README.md)
   - `DEPLOY.md`
4. **Commit changes**

### Opção 2: Usar Interface Web
1. **Abra**: `upload_web.html` no navegador
2. **Cole o token**: `SEU_TOKEN_AQUI` (gerado em https://github.com/settings/tokens)
3. **Clique em Upload**

## 🔧 Ativar GitHub Pages (Manual)

Após upload:
1. **Settings** → **Pages**
2. **Source**: "Deploy from a branch"
3. **Branch**: "main"
4. **Folder**: "/ (root)"
5. **Save**
